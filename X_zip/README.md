## 本文件夹实现了一个基于哈夫曼编码的无损文件压缩/解压小程序X_zip
## 在命令行下，该程序需要四个参数才能正常运行:
+ 程序所在位置，若程序已经在命令行操作的当前目录下，则只需要输入程序名称即可。
+ 指令，有压缩和解压缩两种，若需要压缩文件请输入"zip",若需要解压缩请输入”unzip"，区分大小写。
+ 输出文件名称
+ 目标文件名称
